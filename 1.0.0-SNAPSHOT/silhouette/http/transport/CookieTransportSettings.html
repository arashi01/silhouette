CookieTransportSettings 

final case class CookieTransportSettings(name: String, path: String = "/", domain: Option[String] = None, secure: Boolean = true, httpOnly: Boolean = true, maxAge: Option[FiniteDuration] = None) extends TransportSettings with Product with Serializable

The settings for the cookie transport.

name

The cookie name.

path

The cookie path.

domain

The cookie domain.

secure

Whether this cookie is secured, sent only for HTTPS requests.

httpOnly

Whether this cookie is HTTP only, i.e. not accessible from client-side JavaScript code.

maxAge

The duration a cookie expires. None for a transient cookie.
