case class SimpleConsumerConfiguration(jws: JwsConfiguration[String], requireSubject: Boolean = false, requireJwtID: Boolean = false, requireExpirationTime: Boolean = false, requireIssuedAt: Boolean = false, requireNotBefore: Boolean = false, expectedIssuer: Option[String] = None, expectedAudience: Option[List[String]] = None) extends Product with Serializable

The simple consumer configuration.

jws

The JWS configuration.

requireSubject

Indicates if the 'sub' claim must be set during JWT consumption.

requireJwtID

Indicates if the 'jti' claim must be set during JWT consumption.

requireExpirationTime

Indicates if the 'exp' claim must be set during JWT consumption.

requireIssuedAt

Indicates if the 'iat' claim must be set during JWT consumption.

requireNotBefore

Indicates if the 'nbf' claim must be set during JWT consumption.

expectedIssuer

Indicates the expected value of the issuer ("iss") claim and that the claim is required.

expectedAudience

Set the audience value(s) to use when validating the audience ("aud") claim of a JWT and require that an audience claim be present.
